# 自定义造型 PNG 规范

当前游戏内面板和导入器支持 `body`（身体/胸部）、`head`、`hair`、`beard` 四个图层。

## 文件要求

- 文件必须是 `.png`，带有效 PNG 文件头。
- 必须是带 Alpha 的 RGBA PNG；背景区域必须完全透明（Alpha = 0）。
- 图片必须同时包含可见像素和透明像素；不接受全透明图或整个矩形都不透明的图片。
- 不要让导出工具自动裁切、补边或缩放。画布大小和透明留白都参与定位。
- 每张 PNG 只放一个图层：头部图不带发型/胡子；发型图只放发型；胡子图只放胡子。

## 导入器当前接受的标准画布

| 角色层 | 画布尺寸 | 内容 |
| --- | ---: | --- |
| `head` | 50 × 68 px | 脸和头部基础层，不含头发与胡子 |
| `hair` | 56 × 52 px | 发型层，不含脸 |
| `beard` | 48 × 42 px | 胡子层，不含脸 |
| `body` | 78 × 58 px | 女性身体/胸部基础层，不含头、发型、胡子和护甲 |

这些尺寸是导入器的严格校验值。导入器会复用对应样例的 Battle Brothers brush 定位参数；尺寸不一致会被拒绝，而不是自动缩放。

预置的女性发型和 `Female body 01–02` 都是从 FantasyBro 资源包提取的特殊原版 brush，保留了各自的非标准定位和尺寸；它们是内置预置，不是自定义导入的尺寸模板。自定义 `body` 导入以 `Female body 01` 的 78 × 58 画布和坐标为模板。

## 命名与显示

- 显示名称最多 48 个可打印 ASCII 字符。当前注入式游戏 UI 对中文等非 ASCII 标签的显示不可靠。
- ID 由工具自动生成，例如 `bbca_head_02`，不要手工复用已有 ID。

## 导入流程

运行 `Manage-Custom-Appearance-Assets.bat`，在管理器中选择 PNG 和角色层并执行验证导入。
需要写入游戏时，确认 Battle Brothers 已关闭，再使用管理器中的构建/安装动作；关闭管理器
本身不会自动安装或启动游戏。进入存档后按 `Shift+X`，在 Custom Appearance 面板中选择
新资源并点击 APPLY。
